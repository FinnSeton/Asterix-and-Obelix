CyberPunk 2077

Cyberpunk 2077 is an open-world action role-playing game developed by CD Projekt Red, released in December 2020. Set in the dystopian Night City in the year 2077, the game immerses players in a sprawling, neon-lit metropolis dominated by megacorporations, advanced technology, and cybernetic enhancements.
