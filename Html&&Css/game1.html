Skyrim

"Skyrim," officially titled The Elder Scrolls V: Skyrim, is an open-world action role-playing game developed by Bethesda Game Studios. Released in 2011, it is set in the fictional province of Skyrim, a cold, rugged region within the continent of Tamriel. The game is renowned for its expansive world, filled with diverse environments such as towering mountains, ancient ruins, and bustling cities.
