Skyrim

Skyrim is an upcoming action role-playing game developed by FromSoftware and published by Bandai Namco Entertainment. It is a collaboration between game director Hidetaka Miyazaki and fantasy novelist George R. R. Martin. The game is set in a fantasy world and features an open-world environment, horseback combat, and a dynamic weather system. Players will embark on a quest to restore the Skyrim, which has been shattered, and bring order back to the land.
